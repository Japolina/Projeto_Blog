## ♻️ Como Contribuir

```bash
  # Clone the project
  $ git clone https://github.com/Japolina/Projeto_Blog.git
```

```bash
  # Enter directory
  $ cd Projeto_Blog
```

```bash
  # Install the dependencies, if you use npm
  $ npm install
```

<p align="center">ou</p>

```bash
  # Install the dependencies, if you use yarn
  $ yarn
```
